User Interface VS User Experience
Apa itu User Interface Design?

UI Design adalah proses membuat antarmuka dalam perangkat lunak atau perangkat yang terkomputerisasi dengan fokus pada penampilan atau gaya. Desainer bertujuan untuk membuat desain yang menurut pengguna mudah digunakan dan menyenangkan.

Apa itu User Experience Design?

UX Design adalah proses menciptkan produk yang memberikan pengalaman bermakna dan relevan bagi pengguna. ini melibatkan seluruh proses desain untuk memperoleh dan mengintegrasikan produk, termasuk aspek branding, desain, kegunaan, dan fungsi
